# iHub Africa Website Prototype

## Project Overview

This project is developed as the Sprint 5 Capstone Project. The objective is to design and build a responsive website prototype for iHub Africa that reflects its mission, services, and impact on young people pursuing careers in the digital economy.

The project demonstrates our understanding of web development fundamentals, responsive design principles, teamwork, Git collaboration workflows, and professional project documentation.

## About iHub Africa

iHub Africa is an innovation hub that empowers young people with technical and professional skills required to succeed in the digital economy. Through training programmes, mentorship, and practical projects, iHub Africa helps participants become job-ready and prepared for freelance opportunities.

Our website prototype was designed to communicate iHub Africa's mission, showcase its programmes, and provide visitors with an easy way to learn more about the organisation.

## Git Collaboration Workflow

To ensure effective collaboration, the team followed a structured Git workflow:

1. Created a shared GitHub repository.
2. Used feature branches for development.
3. Created Pull Requests before merging changes.
4. Conducted code reviews before approval.
5. Used descriptive commit messages.
6. Merged approved changes into the main branch.

Example commit messages:

* Added homepage hero section
* Created responsive navigation menu
* Implemented contact form validation
* Fixed mobile layout issues
